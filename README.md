# 🎬 Movie Ratings Prediction & Insights

## 📌 Overview
This project analyzes movie data to understand the key factors that influence movie ratings and revenue. It uses data analysis techniques, statistical testing, feature engineering, and dimensionality reduction (PCA) to extract meaningful insights.

---

## 🎯 Objective
The main goal of this project is to explore relationships between movie features such as:
- Budget
- Revenue
- Popularity
- Runtime
- Vote count
- Genres

and understand how they affect movie ratings and success.

---

## 📊 Dataset
The dataset includes movie metadata such as:
- Budget
- Revenue
- Release date
- Runtime
- Popularity
- Vote average (rating)
- Vote count
- Genres

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Plotly
- Streamlit

---

## 🔄 Project Workflow
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering (Profit, ROI, Log Scaling)
- Feature Selection (Correlation, Lasso, RFE)
- Statistical Analysis (t-tests, ANOVA, Chi-square)
- Dimensionality Reduction (PCA)
- Interactive Streamlit Dashboard

---

## 📈 Key Insights
- Movie ratings are more influenced by audience engagement (votes & popularity) than budget.
- High budget does not guarantee high ROI.
- Some genres consistently perform better than others.
- PCA reveals hidden patterns between movie success factors.
